Starting Strength Calculator

A basic warm-up calculator for Starting Strength

This is an application made for those following Rippetoe's Starting Strength weight lifting program. The goal is to make sure the user is properly warmed up before doing working sets to minimize the chance of injury. The app was designed to be minimal so it loads fast in the gym.

Features

  • Calculate Warm-up reps per Exercise
  • Show Workout of the Day
  • Rank your strength based on weight/gender
  • Ability to add custom plates

Built With

  • JQuery
  • JQueryUI
  • JQueryMobile
Made : Nov 2014

EBay Price Trend Visualizer

A tool view price trends on eBay

This is an application made to display price and how it changes over a period of time. It also gives basic statistics such as max , min, and average prices.

Features

  • Show average item price by day
  • Ability to get statistics on user selected group of items
  • Dynamic filter by
    • Name
    • Date
    • Price

Built With

  • JQuery
  • NodeJS
  • AngularJS
  • D3.js
Made : Mar 2015
Github : Ebay Averagulizer
