Starting Strength

A basic warm-up calculator for Starting Strength

This is an application made for those following Rippetoe's Starting Strength weight lifting program. The goal is to make sure the user is properly warmed up before doing working sets to minimize the chance of injury. The app was designed to be minimal so it loads fast in the gym.

Features

  • Calculate Warm-up reps per Exercise
  • Show Workout of the Day
  • Rank your strength based on weight/gender
  • Ability to add custom plates

Built With

  • JQuery
  • JQueryUI
  • JQueryMobile
